Description:
myTimes time/task tracking is an easy to use iPhone/iPod Touch application for project time tracking. The captured times are organized in a project-sub project structure. In addition to track the start and end time (duration) you can also track pause times and add a description for each task. The project and sub project lists displays the accumulated timess of its items. myTimes allows you to export your tracked times and the accumulations to an excel friedly CSV format via mail export.
Features:
* track of project task times
* remarks for the single tasks/times
* recording times using start/stop mechanism
* track pause time
* eMail CSV or XML export
* XML import (e.g. for backup/restore)
* available in german and english
* calendar view
